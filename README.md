# Generador de Contraseñas

Este proyecto es un generador de contraseñas en Python que crea contraseñas aleatorias seguras usando letras, números y símbolos.

## Características

- Solicita al usuario la longitud deseada de la contraseña.
- Genera contraseñas aleatorias con letras mayúsculas, minúsculas, dígitos y caracteres especiales.
- Muestra la contraseña generada por pantalla.

## Requisitos

- Python 3.x
- No requiere dependencias externas adicionales.

## Uso

1. Abre una terminal.
2. Navega a la carpeta del proyecto:
3. Ejecuta el script:

```bash
python password_gen.py
```

4. Ingresa la longitud de la contraseña cuando se te solicite.
5. El programa mostrará una contraseña generada aleatoriamente.

## Ejemplo

```text
Ingrese la longitud de la contraseña: 12
Contraseña generada: xK3@f9!Lm2#Q
```

## Mejoras sugeridas
